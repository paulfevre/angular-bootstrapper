# Angular Bootstrapper

Angular & Angular Material application bootstrapper

## Requirements

- Node/NPM

## Install

Install NPM dependencies

```
npm install
```

## Usage

Launch integrate web server.

```
npm run start
```

Go to [http://localhost:4200](http://localhost:4200).
